Java nedir?

Java, 1995 yılında oluşturulan popüler bir programlama dilidir.
Oracle'a aittir ve 3 milyardan fazla cihaz Java çalıştırmaktadır.
İçin kullanılır:

  • Mobil uygulamalar (özellikle Android uygulamaları)
  • Masaüstü uygulamaları
  • Web uygulamaları
  • Web sunucuları ve uygulama sunucuları
  • Oyunlar
  • Veritabanı bağlantısı
  • Ve çok daha fazlası!

Neden Java'yı Kullanmalı?

  • Java farklı platformlarda çalışır (Windows, Mac, Linux, Raspberry Pi vb.)
  • Dünyanın en popüler programlama dillerinden biridir
  • Mevcut iş piyasasında büyük bir talep var
  • Öğrenmesi kolay ve kullanımı basittir
  • Açık kaynaklı ve ücretsizdir
  • Güvenli, hızlı ve güçlüdür
  • Çok büyük bir topluluk desteği var (on milyonlarca geliştirici)
  • Java, programlara net bir yapı kazandıran ve kodun yeniden kullanılmasına izin vererek geliştirme maliyetlerini düşüren nesne yönelimli bir dildir.
  • Java , C++ ve C#' a yakın olduğundan programcıların Java'ya veya Java'ya geçişini kolaylaştırır.


Sisteminize 64 bit Java yükleme

  1. 64 bit Windows çevrimdışı indirmeyi seçin. Dosya İndirme iletişim kutusu görüntülenir.
  2. Klasör konumunu seçin. (Bilgisayarınızda kolayca bulabileceğiniz bir konumu, örneğin masaüstünü tercih edin) ve Kaydet'e tıklayın.
  3. Tarayıcıyı ve diğer tüm uygulamaları kapatın.
  4. Kaydettiğiniz dosyanın simgesine çift tıklayarak yükleme işlemini başlatın.
